# テーブル仕様

## レース情報

`scrape_race_info()`関数が返すDataFrameの仕様。RACE_INFO_COLUMNSに定義された25カラムを持つ1行のDataFrame。

| カラム名     | 型    | 説明                             | 例               |
|--------------|-------|----------------------------------|------------------|
| レースID     | str   | netkeibaのレースID（12桁）        | "202505021211"   |
| レース名     | str   | レース名                          | "日本ダービー"   |
| 発走時刻     | str   | 発走時刻（HH:MM形式）              | "15:40"          |
| 天候         | str   | 天候（不明時は空文字列）            | "晴", "曇", "雨" |
| 馬場         | str   | 馬場状態（不明時は空文字列）        | "良", "稍", "重", "不" |
| 芝ダ         | str   | 芝・ダート・障害の種別             | "芝", "ダ", "障" |
| 距離         | int   | 距離（メートル）                   | 2400             |
| 左右         | str   | コースの回り方（該当なしは空文字列） | "左", "右", "直" |
| コース       | str   | コース分類（該当なしは空文字列）    | "A", "B", "C", "D" |
| 内外         | str   | 内回り・外回り（該当なしは空文字列） | "内", "外", "外-内" |
| 競馬場       | str   | 競馬場名                          | "東京", "中山"   |
| 回           | int   | 開催回（不明時は0）                | 2, 5             |
| 開催日       | int   | 開催日目（不明時は0）              | 12, 8            |
| 競争種別     | str   | 競争種別（条件戦の場合）            | "サラ系３歳以上", "サラ系２歳" |
| 競争条件     | str   | 競争条件                          | "オープン", "２勝クラス", "新馬" |
| グレード     | str   | グレード（オープン以外は空文字列）   | "G1", "G2", "G3", "L", "OP", "JG1" |
| 競走記号     | str   | 競走記号（括弧付き、なしは空文字列） | "(国際)(指)", "(混)[指]" |
| 重量種別     | str   | 斤量条件                          | "定量", "別定", "ハンデ", "馬齢" |
| 頭数         | int   | 出走頭数（不明時は0）              | 18, 12           |
| 1着賞金      | int   | 1着賞金（万円、不明時は0）          | 30000            |
| 2着賞金      | int   | 2着賞金（万円、不明時は0）          | 12000            |
| 3着賞金      | int   | 3着賞金（万円、不明時は0）          | 7500             |
| 4着賞金      | int   | 4着賞金（万円、不明時は0）          | 4500             |
| 5着賞金      | int   | 5着賞金（万円、不明時は0）          | 3000             |
